About DIGGER


  • DIGGER was jointly developed by the groups Big Data in Biomedicine and Computational Systems Medicine.

  • DIGGER was created with funding from BMBF Sys_CARE project
  • Source code is available on Github

  • We build DIGGER using:   Django, NetworkX, Vis.js, DNA Features Viewer.
